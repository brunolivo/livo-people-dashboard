# Livo People Dashboard

**Q1 2026 · People & Performance Hub**

A single-page HR dashboard for Livo Health, built as a self-contained HTML application wrapped in a React/TSX iframe component.

## Contents

| File | Description |
|---|---|
| `index.html` | Full standalone app (open directly in any browser) |
| `Q1PerformanceCycle.tsx` | React wrapper component (iframe srcDoc) |
| `Logos/` | Livo brand assets |
| `design-system.md` | Colour palette, typography, and component guidelines |

## Features

- **OKRs** — Company & Area objectives with editable Key Results
- **Talent Map** — 2×2 performance/values matrix
- **Results** — Individual scores and succession planning
- **Compensation** — Unified people table with salary, ESOP, OTE, history log
- **Admin Panel** — Roles, comp settings (budget + salary bands), team management
- **Login** — Auth flow with forgot-password reset

## Stack

Pure HTML/CSS/JS — no build step required. Open `index.html` in a browser to run locally.
